## Examples

```python
import askmagic

print(askmagic.ask.question("en", "Object Pascal sucks?"))

print("\n")

print(askmagic.ask.answer("en"))
print(askmagic.ask.answer("uk"))
print(askmagic.ask.answer("ru"))
```
